# Bash cmd

## open file in the terminal 

  ``` cat file.js ```
  
## delete file

  * delete a file:

    `rm <file_name>`

  * delete a directory:

    `rm -r <directory_name>`

## File Tree

- `tree`: show only folder structure
- `tree /f` show folders with files

## Curl
- `curl <url>` : make get to url
- `curl -v <url>` : make get to the url with the request info
